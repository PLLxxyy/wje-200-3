# 随份子礼金簿

一款轻量级人情往来礼金记录工具，帮你记住每一份随礼。

## 功能

- **记录管理**：新增/删除礼金记录，记录对方姓名、场合、金额、日期、收/送类型
- **时间倒序**：按日期倒序展示所有记录
- **搜索筛选**：支持按姓名搜索，按场合（婚礼/满月/乔迁/生日/丧事/其他）筛选
- **收支统计**：总收到、总送出、净收支一目了然
- **按人汇总**：按人名汇总往来总额
- **未还礼提醒**：高亮标记送过礼但对方未回礼的人
- **本地存储**：数据保存在浏览器 localStorage，刷新不丢失

## 技术栈

- Vite + React 18 + TypeScript
- localStorage 数据持久化
- 纯 CSS（样式写在 index.html 的 `<style>` 标签中）
- 无第三方 UI 库

## 快速开始

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:5200 即可使用。

## 构建

```bash
npm run build
```

构建产物在 `dist` 目录。
